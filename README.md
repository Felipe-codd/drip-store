# drip-store
Projeto front-end de loja virtual desenvolvido com React

👟 Projeto Drip Store | Geração Tech

Este projeto representa um marco importante na minha jornada como desenvolvedor. Desenvolvido durante o curso Full Stack da Geração Tech, a Drip Store é muito mais do que um simples e-commerce de calçados; é o resultado prático de semanas de imersão em ecossistemas modernos de desenvolvimento web.

A Proposta

A ideia central foi construir uma plataforma de vendas robusta e visualmente atraente, simulando um ambiente real de mercado. Através deste projeto, pude explorar como uma interface de usuário (UI) bem pensada pode guiar a experiência de compra, desde a navegação fluida na Home até o refinamento de busca na listagem de produtos.

O Que Há Por Baixo do Capô?

Para dar vida à loja, utilizei o React.js como coração da aplicação, aproveitando a componentização para manter o código limpo e escalável. A navegação entre as páginas acontece de forma instantânea graças ao React Router, proporcionando aquela sensação de "Single Page Application" que os usuários adoram.

A estilização foi um desafio à parte: o foco foi garantir que a identidade visual da Drip Store fosse respeitada, mantendo a responsividade total para que a experiência seja impecável tanto em um monitor 4K quanto na tela de um celular.

Aprendizado e Créditos

Este software foi desenvolvido com o suporte e a metodologia da Geração Tech, que forneceu as diretrizes e o protótipo base para que pudéssemos exercitar o pensamento lógico e a arquitetura de componentes. Foi um exercício intenso de transformar um design estático em uma aplicação funcional e interativa.
